本项目简介
	该项目为提供联通平台所有发短信功能包括（定时短信、时时短信都基本功能）同时为手机客户端、家校E通、校务通等平台提供短信基础功能服务。
本项目技术架构设计
	采用主流的REST面向服务面向组件的架构设计思想下采用SpringMVC的REST技术实现规范的实现。
	SpringMVC+Hibernate+SQLSERVER2005+Maven+Git(版本控制)
